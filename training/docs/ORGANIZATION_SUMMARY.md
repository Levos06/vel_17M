# Резюме организации проекта

## Что было сделано

### 1. Создана структура папок
- `data/datasets/` - для всех датасетов
- `data/results/` - для результатов обработки данных
- `models/` - для сохраненных моделей
- `outputs/embeddings/` - для эмбеддингов
- `outputs/visualizations/` - для визуализаций
- `scripts/` - для вспомогательных скриптов

### 2. Организованы данные
- Перемещены датасеты: `stanford/`, `stanford_train/`, `it/` → `data/datasets/`
- Перемещены результаты: `tsne_results/`, `tsne_results_10k/` → `data/results/`
- Перемещены файлы данных: `*.parquet`, `*.bin` → `data/datasets/`
- Перемещены sample файлы: `filtered_samples.txt`, `random_samples.*` → `data/results/`

### 3. Организованы модели и чекпоинты
- Перемещена модель: `academic_filter_model/` → `models/`
- Объединены чекпоинты: `checkpoints_sft/`, `checkpoints_sft_01/` → `checkpoints/`

### 4. Организованы логи
- Объединены логи: `logs_sft/`, `logs_sft_01/`, `logs_v2/` → `logs/`

### 5. Организованы выходные данные
- Перемещены эмбеддинги: `embeddings/` → `outputs/embeddings/`
- Перемещены визуализации: `tsne_interactive_plot.html` → `outputs/visualizations/`

### 6. Организованы скрипты
- Перемещен скрипт: `logs/loss.py` → `scripts/`

### 7. Очистка
- Удалены временные файлы: `__pycache__/`, `.DS_Store`
- Создан `.gitignore` для игнорирования временных файлов

### 8. Документация
- Обновлен `README.md` с новой структурой проекта
- Создан `PROJECT_STRUCTURE.md` с подробным описанием структуры

## Новая структура

```
transformer/
├── data/
│   ├── datasets/       # Все датасеты
│   └── results/        # Результаты обработки
├── models/             # Сохраненные модели
├── checkpoints/        # Все чекпоинты (объединены)
├── logs/               # Все логи (объединены)
├── outputs/
│   ├── embeddings/     # Эмбеддинги
│   └── visualizations/ # Визуализации
├── scripts/            # Вспомогательные скрипты
└── [скрипты Python]    # Основные скрипты в корне
```
